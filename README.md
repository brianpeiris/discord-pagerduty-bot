# discord-pagerduty-bot

> Note: This repo is archived and the bot code will likely need updating to conform to Discord's latest API.

A discord bot that lets you trigger and inpsect PagerDuty.

Needs to be configured with the following environment variables:

PAGERDUTY_SERVICE  
PAGERDUTY_SCHEDULE  
PAGERDUTY_ACCESS_TOKEN  
PAGERDUTY_INTEGRATION_KEY  
DISCORD_TOKEN  
DISCORD_CHANNELS
